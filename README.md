# GCRH Engenharia — website institucional

Website institucional estático da **GCRH Engenharia LTDA**, desenvolvido para apresentar serviços técnicos especializados em engenharia consultiva, recursos hídricos, saneamento, meio ambiente, geoprocessamento, modelagem computacional, treinamentos e soluções digitais aplicadas.

O site foi estruturado para publicação via **GitHub Pages**, com foco em carregamento rápido, manutenção simples, navegação responsiva e expansão progressiva por meio de atualizações semanais e mensais.

## Objetivo do website

O website tem como finalidade:

- Apresentar a GCRH Engenharia como empresa técnica especializada.
- Divulgar serviços de engenharia, meio ambiente, recursos hídricos e saneamento.
- Facilitar a solicitação de orçamento por potenciais clientes.
- Organizar páginas específicas para serviços, treinamentos, materiais técnicos e conteúdos institucionais.
- Reforçar credibilidade técnica por meio de textos explicativos, imagens, estudos de caso, formulários e canais de contato.
- Servir como base para futuras soluções digitais, aplicações web e automações técnicas.
