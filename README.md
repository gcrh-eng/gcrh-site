# GCRH Engenharia - website estático

Website institucional estático da GCRH Engenharia, estruturado para GitHub Pages.

## Teste local

No VS Code, use a extensão **Live Server** e abra `index.html`.

Alternativa com Python:

```bash
python -m http.server 8000
```

Acesse `http://localhost:8000/`.

## Publicação

Use GitHub Desktop para commit e push na branch `main`.

## Observações

- Os PDFs em `assets/docs/cartilhas/` são placeholders e devem ser substituídos pelos materiais finais.
- As imagens originais do template Coefficient foram mantidas como placeholders e devem ser substituídas por fotografias próprias da GCRH.
- O template foi reconstruído em HTML5/CSS responsivo, preservando referência/crédito ao template Coefficient (TEMPLATED), conforme licença CC BY 3.0.
